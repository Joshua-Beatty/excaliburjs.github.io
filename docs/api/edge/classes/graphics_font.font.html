quality

quality: number = 2
  • Defined in src/engine/Graphics/Font.ts:60

Font quality determines the size of the underlying rastered text, higher quality means less jagged edges. If quality is set to 1, then just enough raster bitmap is generated to render the text.

You can think of quality as how zoomed in to the text you can get before seeing jagged edges.

(Default 4)
