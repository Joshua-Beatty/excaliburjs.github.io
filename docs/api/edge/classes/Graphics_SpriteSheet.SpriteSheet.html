Static fromImageSource

  • fromImageSource(options: SpriteSheetGridOptions): SpriteSheet
    • Defined in src/engine/Graphics/SpriteSheet.ts:166

    Create a SpriteSheet from an ImageSource organized in a grid

    Example:

    const spriteSheet = SpriteSheet.fromImageSource({
  image: imageSource,
  grid: {
    rows: 5,
    columns: 2,
    spriteWidth: 32, // pixels
    spriteHeight: 32 // pixels
  },
  // Optionally specify spacing
  spacing: {
    // pixels from the top left to start the sprite parsing
    originOffset: {
      x: 5,
      y: 5
    },
    // pixels between each sprite while parsing
    margin: {
      x: 1,
      y: 1
    }
  }
})

    Parameters

    • options: SpriteSheetGridOptions

    Returns SpriteSheet
